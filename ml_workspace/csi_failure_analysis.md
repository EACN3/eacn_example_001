## 1. CSI失败的理论诊断

### 1.1 为什么50 epoch的CSI比Harmony更差？

**根本原因：对比学习的"uniformity-alignment"二元张力。**

InfoNCE损失有两个隐含目标（Wang & Isola, 2020）：
- **Alignment**：正样本对（MNN对）被拉近
- **Uniformity**：所有嵌入在超球面上均匀分布（负样本被推远）

问题在于**uniformity项**：它将所有非MNN对的细胞推远，包括同类型但不是MNN的细胞。

具体来说：
- T cell cluster A（batch 1）和 T cell cluster A（batch 2）通过MNN被拉近 ✓
- 但 T cell cluster A 和相邻的 T cell cluster B 没有MNN关系 → 被uniformity推远 ✗
- 这破坏了T cell亚群间的连续过渡结构

**对稀有亚群的影响**：
- 稀有亚群没有MNN对 → 不受alignment力
- 但受uniformity力 → 被推到超球面的"空旷区域"
- 这不是保持原位，而是被推到一个无意义的位置

### 1.2 InfoNCE是否天然破坏过渡态？

**是的。**InfoNCE的负样本机制将非正样本对一律视为"不同的"，但生物学中细胞状态是连续的——naive T → effector T → exhausted T 是一个梯度。InfoNCE会在梯度上的相邻状态之间施加排斥力，打断连续过渡。

### 1.3 L_structure为什么没救回来？

L_structure（批次内邻域保持）只保证同批次内的局部结构，但跨批次的结构被InfoNCE破坏了。而且如果L_structure权重太大，又会阻止正常的批次校正。这是一个无法平衡的trade-off。

### 1.4 核心教训

**对比学习适合分类问题（每个样本属于一个明确的类），不适合保结构问题（样本之间有连续的拓扑关系）。**

单细胞整合需要保结构——对比学习是错误的工具。

## 2. MNN图消息传递 vs 对比学习

### 2.1 MNN Laplacian平滑的本质

```
z_i^{new} = (1-α) * z_i + α * mean(z_j for j in MNN_neighbors(i))
```

这是在MNN图上做一步**加权平均/扩散**：每个细胞向其MNN邻居的方向移动。

**优势（vs 对比学习）**：

| 特性 | MNN消息传递 | 对比学习(CSI) |
|------|-------------|---------------|
| 对无MNN细胞的影响 | **零**（不在图上=不移动） | 有（uniformity推远） |
| 连续过渡保持 | ✓（平滑只影响直接连接） | ✗（uniformity破坏） |
| 超参数 | α（步长）仅一个 | τ, λ, lr, epochs等多个 |
| 收敛保证 | ✓（Laplacian平滑收敛性已证明） | 不保证 |
| 训练需求 | 无（闭式解或迭代平均） | 需GPU训练 |
| 效率 | O(n×k×iterations) ≈ 分钟级 | O(n×epochs×batch) ≈ 十分钟级 |
| 表达能力 | 线性（均值平滑） | 非线性（MLP） |

### 2.2 MNN消息传递的理论保证

Laplacian平滑在图信号处理中有严格保证：
- 收敛到MNN连通分量的均值
- 连通分量内的方差单调下降
- 不影响不连通的分量

对选择性整合的意义：
- MNN连通的细胞（shared）→ 逐步对齐
- MNN不连通的细胞（unique）→ 完全不动
- 对齐速度由MNN连接密度决定 → BD天然内建

### 2.3 MNN消息传递的局限

1. **只能做线性平滑**：如果批次效应是非线性的（如不同批次的PCA方向不同），线性平均可能不够
2. **过平滑风险**：迭代太多会让所有连通细胞坍缩到同一点
3. **MNN质量依赖初始空间**：如果PCA空间被批次效应严重扭曲，MNN匹配可能不准

### 2.4 结论

**MNN消息传递是选择性整合的更好工具**——简单、有保证、不破坏结构。它的局限（线性、过平滑）可以通过控制迭代次数和使用BD加权来缓解。

## 3. 超网络方案（如果MNN平滑不够好）

### 3.1 核心思想

用超网络为每个细胞生成**个性化的校正向量**，校正强度由细胞的局部上下文决定。

### 3.2 架构

```
输入特征（per-cell context vector）：
  c_i = [BD(i),                    # batch diversity (标量)
         mean_expr_neighbors(i),   # 邻域平均表达（50维PCA）
         batch_one_hot(i),         # 批次编码（B维）
         local_density(i)]         # 局部密度（标量）
  → 总维度 ≈ 52+B

超网络 H_φ：
  H_φ(c_i) → δ_i ∈ R^d_latent    # 校正向量

校正后嵌入：
  z_i^corrected = z_i^raw + BD(i) * H_φ(c_i)

  注意：乘以BD(i)保证独有细胞（BD≈0）的校正向量被归零
```

### 3.3 训练信号

**自监督**：用MNN对作为训练信号——校正后的MNN对应该比校正前更近：

```
L_hyper = Σ_{(i,j)∈MNN} ||z_i^corrected - z_j^corrected||²
        + λ * Σ_i ||δ_i||²  # L2正则化防止过大校正
```

**不需要标签**——MNN对本身就是"应该对齐"的信号。

### 3.4 与MNN平滑的关系

超网络是MNN平滑的非线性推广：
- MNN平滑：δ_i = α * (mean(z_j for MNN neighbors) - z_i)  ← 固定的线性规则
- 超网络：δ_i = H_φ(c_i)  ← 学习的非线性规则

如果批次效应是线性的，超网络会退化到MNN平滑。如果是非线性的，超网络能学到更好的校正。

### 3.5 效率

```
超网络参数量：MLP [52+B → 128 → 64 → 30] ≈ 15K参数
训练：只需在MNN对上训练，数据量 = |MNN pairs| ≈ 100K-1M
→ 训练时间 < 5分钟
推理：每个细胞一次前向传播 → O(n) ≈ 秒级
总时间：PCA(2min) + kNN(2min) + MNN(1min) + 训练(5min) + 推理(1s) ≈ 10分钟
```

### 3.6 优先级建议

1. **先试MNN Laplacian平滑**（最简单，有理论保证）
2. 如果线性不够 → **加BD加权的MNN平滑**（BD控制步长）
3. 如果仍不够 → **超网络**（非线性校正，可学习）

每一层都比上一层更强但更复杂，按需升级。
