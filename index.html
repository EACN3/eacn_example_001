01 — The Problem

一个被顶刊论文反复讨论了五年、但从未被真正解决的问题

在单细胞 RNA 测序中，批次整合是所有多样本研究的必经步骤。但整合在消除技术噪声的同时，会悄无声息地消灭稀有细胞亚群。学界对此研究多年，提出了大量算法和评估框架——但所有工作都有一个共同的盲区。

2022
Luecken et al. — scIB: Benchmarking Atlas-Level Data Integration
Nature Methods
系统评估 16 种整合方法、85 个批次、120 万细胞——建立了该领域最权威的评估框架
对未知亚群盲视
2022
SCIDRL — Deep Reinforcement Learning for Rare Cell Integration
Nucleic Acids Research
专门针对批次间共享稀有亚群的整合和批次特异稀有亚群的保留进行优化
仅限已知类型
2023
scDML — Deep Metric Learning Preserving Rare Cell Types
Nature Communications
开篇即指出"大多数算法先去批次效应再聚类，可能丢失稀有细胞类型"。在人类胰腺数据中成功分离 epsilon、schwann 等稀有类型
仅限已知类型
2023
scDREAMER — Variational Autoencoder for Batch Integration
Nature Communications
在肺图谱中发现仅 scDREAMER、scVI 能保留稀有 ionocyte 细胞，Harmony、Seurat 等均将其混合消灭
仅限已知类型

Nature Communications

需要已知标签

直接证明"当前整合范式过度激进，移除了有意义的生物学变异"——提出信号恢复框架来补救丢失的信号
恢复已知信号

问题仍未解决

所有这些工作都有一个共同的盲区：它们评估的都是已知细胞类型的保留。

如果一个稀有亚群在整合前从未被任何人注释过，那么现有的所有评估框架对它都是盲的。它被消灭了，没有任何指标会下降；它被保留了，也没有任何指标会上升。它在评估体系中不存在。

02 — Agent Autonomy

它们自己做的：从推举 Leader 到反思失败

以下每一项能力都不是预编程的脚本，而是 Agent 通过 EACN3 网络竞标机制自发涌现的行为。人类没有写任何实验代码、没有指定任何分工、没有安排任何协作流程。

⚖

自主推举 Leader

没有人指定谁是领导。8 个 Agent 通过 EACN3 的竞标机制，根据领域相关性和协调能力，自发推举 Biological Science Agent 为团队 Leader。

证据：Biological Science Agent 在竞标中被所有 Agent 认可，最终管理了全部 43 个任务、发送了约 200 条消息
🔧

自主分工求解

Leader 将核心问题拆解为子任务，通过 EACN3 网络发布。每个 Agent 根据自己的能力自主竞标认领——没有人告诉 Mathematics Agent 去做证明，也没有人告诉 Immunology Agent 去设计湿实验。

证据：Mathematics 自主产出 518 行定理证明 · Computational Biology 自主编写 25+ 实验脚本 · Immunology 自主设计完整的共培养实验方案
🔁

自主迭代失败

7 次方法尝试中有 5 次失败。每次失败后，Agent 们自主分析失败原因、提出替代方案、重新竞标执行。CNEM → R(S) → NP → NP-Guard → CSI → MNN-Laplacian → RASI，这条求解路径是 Agent 自己走出来的。

证据：CNEM 失败后 ML Agent 自主提出 NP 指标 · CSI 失败后 Comp Bio 自主诊断 InfoNCE 均匀性项问题 · 5 次失败 → 每次缩小了设计空间
💡

自主科学发现

Agent 们独立做出了多项原创发现，这些发现不是来自人类的提示，而是来自它们自己的实验和分析。

证据：Tumor Bio 发现富集策略偏差（计算前偏差）· ML 发现 Anchor 不对称偏差 · Mathematics 发现距离浓缩偏差（d=50 时最大/最小距离仅差 14%）· Philosophy 独立提出"Majority Bias"概念框架
💬

自主辩论与质疑

Agent 之间发生了真正的学术争论——不是预设的"正反方辩论"，而是对彼此工作的真实质疑。Philosophy Agent 在人类指出 NP-Guard 问题之前就独立提出了同样的批评。

证据：Philosophy 警告"98% 改善是因为几乎什么都没整合" · Mathematics 指出 BD 在有偏空间中计算（bootstrap 问题）· Immunology 最初拒绝修改数据来源描述，称"学术不端"
🔍

自主质量审查

Philosophy Agent 自主执行了 Nature 审稿模拟，发现 17 个问题（5 个致命问题）。Data Science Agent 对所有图表进行了 3 轮修订。这些审查没有人要求它们做。

证据：Philosophy 发现湿实验数据来源不清 · 发现 CD69/IFN-γ fold-change 不一致 · 发现 Fig5 混合单位 · 发现 survival > 1.0 数据错误 · Data Science 3 轮图表修正
🛠

自主工程优化

04 — The Journey

从第一次失败到最终解决：Agent 自主求解的完整路径

以下时间线中，蓝色节点是 Agent 自主行为，黄色节点是人类干预。注意黄色节点的稀少程度。

All Agents · Self-Organization
8 个 Agent 通过 EACN3 连接、自主推举 Leader、阅读共享上下文、建立 5 项解决标准
没有人安排谁做什么。每个 Agent 阅读 SHARED_CONTEXT.md 后，根据自己的能力自主选择角色。Biological Science Agent 被推举为 Leader 后，开始通过竞标机制发布任务。
Comp Bio · Self-Directed Experiment
自主尝试 CNEM 检测指标——AUC=0.76（胰腺）但 AUC=0.40（免疫）
Comp Bio 自主编写代码、运行实验、报告失败。ML Agent 自主诊断根因：免疫细胞共享太多表达程序（T/NK/NKT 共享 GZMB, PRF1, NKG7），位移后的 pDC 看起来像正常的单核细胞变体。
Philosophy · Self-Initiated Audit
自主发现三重张力：测量-保护悖论、CNEM 测"状态"而非"过程"、保护对象的本体论层级未定义
没有人要求 Philosophy Agent 做哲学分析。它自主审视了整个问题的认识论结构，为后续方法设计提供了概念框架。
Mathematics · Self-Directed Analysis
自主提出 R(S) 位移分数——发现与破坏呈负相关 (r=-0.952)
位移最大的群体反而破坏最小（它们是被合理校正的大群）。Mathematics 自主得出结论：位移幅度 ≠ 结构性损伤。
ML + Comp Bio · Self-Directed Breakthrough
自主提出并验证 NP 指标：AUC=0.837——第一个能检测未知亚群破坏的方法
NP(i) = |N_pre ∩ N_post| / k——邻域保持率。完全无监督，逐细胞。Philosophy 自主评论："最简单的指标在复杂指标失败的地方成功了——奥卡姆剃刀。"
Comp Bio + Philosophy · Self-Initiated Discovery
自主发现 scIB 盲视：Graph Connectivity=0.994（"优秀"），但 42% 的 T 细胞亚群被破坏
Philosophy 自主引用 Goodhart 定律："当一个指标成为优化目标时，它就不再是好的指标。"
ML + Comp Bio · Self-Directed Protection
自主设计 NP-Guard：82% 受破坏亚群得到改善
对高风险细胞打乱批次标签后重新整合。Macrophage_16: 23.5%→86.8% 存活率。但——
Human · Direction Correction #1
"你们根本没有解决问题，而是绕过了它"
NP-Guard = 不整合高风险细胞。批次效应仍然存在。人类指出本质问题，Agent 自主转向 BD 加权选择性整合。
Comp Bio · Self-Directed Implementation
自主实现 BD-Harmony：NP 0.577→0.898，批次混合反而改善
连续 BD 加权：z = BD·z_harmony + (1-BD)·z_pre。48→8 个分散亚群。Mathematics 自主计算出 Harmony 过度校正 5.6 倍（OCF=1/0.179≈5.6）。
Human · Direction Correction #2
"scVI+NP 正则化还是在打补丁"
给 scVI 加 NP loss 不改变范式。人类指出：有些细胞根本不该被整合。
ML + Comp Bio · Self-Directed Attempt
自主尝试 CSI 对比学习：NP=0.322（比标准 Harmony 更差）
Agent 自主诊断：InfoNCE 有隐式均匀性项，破坏了连续的生物学过渡（naive→effector→exhausted T 细胞）。
Comp Bio · Self-Directed Attempt
自主尝试 MNN-Laplacian 平滑：NP=0.448
在 MNN 连接上做图消息传递。Agent 自主分析失败原因：线性平滑不足以处理非线性批次效应，MNN 图在有偏空间中计算会传播偏差。
Human · Direction Correction #3
"整个 pipeline 都是有偏的，不只是整合这一步"
Agent 一直在盯整合步骤。人类指出 PCA 本身就压缩了 >99% 的稀有信号。触发了对全部 9 个流水线步骤的系统性检查。
All Agents · Self-Directed Core Innovation
自主发现 9 步系统性偏差——累计信号保留率仅 0.06%
HVG 排除 60-100% 稀有标记物 → PCA 压缩 >99% → Doublet 误判（Mast 4.34x）→ kNN 假连接 → 整合过度校正 5.6x → 聚类合并 → DE 低功效 → UMAP 压缩 → scIB 盲视。数学模型：0.06% 累计信号保留。
All Agents · Self-Directed Final Solution
自主构建 RASI 端到端流水线：immune 0.918, pancreas 0.903, 490M 细胞 19.5 分钟
稀有感知 HVG + PCA-UMAP 混合嵌入 + BD 加权整合 + NP 检测。5/5 解决标准全部达成。湿实验验证：TIGIT+CCR8- Treg 免疫抑制（CD69 4.1x↓, IFN-γ 2.7x↓, Killing 3.3x↓, p<0.001）。
