# 数学智能体 — "聚拢 vs 弥散"形式化定义 (v1)

## 符号约定

设：
- $X^{\text{pre}} = \{x_1, \ldots, x_N\} \subset \mathbb{R}^d$ 为整合前的细胞嵌入（所有批次合并）
- $X^{\text{post}} = \{y_1, \ldots, y_N\} \subset \mathbb{R}^d$ 为整合后的细胞嵌入
- $x_i$ 和 $y_i$ 是同一个细胞在整合前后的表示
- $\mu = \frac{1}{N}\sum_{i=1}^N \delta_{x_i}$，$\nu = \frac{1}{N}\sum_{i=1}^N \delta_{y_i}$ 为对应的经验测度

## 方案 A：基于传输计划的聚拢-弥散指标

### A1. 局部传输熵 (Local Transport Entropy)

对整合前嵌入空间中的任意一个局部邻域 $B(c, r)$（中心 $c$，半径 $r$），定义其中的细胞集合：

$$S = \{i : x_i \in B(c, r)\}$$

这些细胞在整合后的目标位置为 $\{y_i : i \in S\}$。定义 $S$ 中细胞的传输弥散度：

$$D(S) = \frac{1}{|S|} \sum_{i \in S} \|y_i - \bar{y}_S\|^2$$

其中 $\bar{y}_S = \frac{1}{|S|}\sum_{i \in S} y_i$ 是目标位置的质心。

**解读**：
- **聚拢**：$D(S)$ 小 → 邻域内细胞整合后仍然聚在一起（可能与其他批次同类细胞汇聚）
- **弥散**：$D(S)$ 大 → 邻域内细胞整合后被打散到不同位置

### A2. 相对弥散度 (Relative Dispersion)

仅看绝对弥散度不够，需要与"正常"的聚拢行为做对比。定义：

$$R(S) = \frac{D(S)}{D_{\text{ref}}}$$

其中 $D_{\text{ref}}$ 是所有同尺度邻域弥散度的中位数。

- $R(S) \approx 1$：该邻域的行为与大多数邻域一致（正常）
- $R(S) \gg 1$：该邻域异常弥散（稀有亚群被消灭的信号）

### A3. 弥散方向熵 (Directional Entropy of Dispersion)

传输向量 $v_i = y_i - x_i$ 的方向分布也携带信息。对邻域 $S$，将 $\{v_i\}_{i \in S}$ 投影到单位球上，估计其方向分布的熵 $H_{\text{dir}}(S)$。

- **聚拢**：传输向量方向一致（所有细胞朝同一方向移动）→ $H_{\text{dir}}$ 低
- **弥散**：传输向量方向各异（细胞被打散到四面八方）→ $H_{\text{dir}}$ 高

## 方案 B：基于持久同调的拓扑消失指标

### B1. 连通分量持久性变化

对 $X^{\text{pre}}$ 和 $X^{\text{post}}$ 分别计算 0 维持久图：

$$\text{PD}_0^{\text{pre}} = \{(b_j^{\text{pre}}, d_j^{\text{pre}})\}_j, \quad \text{PD}_0^{\text{post}} = \{(b_k^{\text{post}}, d_k^{\text{post}})\}_k$$

一个稀有亚群如果在整合前是一个持久的连通分量（长 bar），整合后该连通分量消失（对应的 bar 缩短或合并），则信号体现为：

$$\Delta\text{PD}_0 = W_p(\text{PD}_0^{\text{pre}}, \text{PD}_0^{\text{post}})$$

Wasserstein-p 距离量化持久图间的总变化。但这是全局指标，需要进一步定位到具体的消失特征。

### B2. 拓扑消失定位

对 $\text{PD}_0^{\text{pre}}$ 中每个持久特征 $(b_j, d_j)$，可以追踪其对应的细胞子集（即该连通分量代表的细胞群）。如果该特征在 $\text{PD}_0^{\text{post}}$ 中没有对应的持久特征，则该细胞群在拓扑上"被消灭"了。

具体做法：用 bottleneck matching 在两个持久图之间建立对应关系。未被匹配到（或被匹配到对角线）的 pre 特征，就是整合中消失的拓扑结构。

## 方案 C：OT + TDA 组合方案

**核心思想**：用 OT 做细粒度诊断（哪些细胞被移动到了哪里），用 TDA 做粗粒度确认（是否有群消失了）。

1. **第一步（TDA 扫描）**：比较整合前后的持久图，标记消失的拓扑特征，定位到对应的细胞子集 $S_{\text{vanished}}$。
2. **第二步（OT 确认）**：计算 $S_{\text{vanished}}$ 的弥散度 $D(S_{\text{vanished}})$ 和方向熵 $H_{\text{dir}}(S_{\text{vanished}})$，确认是否为弥散模式。
3. **第三步（排除假阳性）**：检查弥散是否是"正常整合"的副产物——如果 $S_{\text{vanished}}$ 中的细胞被打散到多个不同大群中（而非汇聚到一个已有大群），则更可能是病理性的。

### 假阳性排除的数学化

定义传输目标集中度：

$$C(S) = \max_k \frac{|\{i \in S : \text{NN}(y_i) \in \text{Cluster}_k\}|}{|S|}$$

其中 $\text{NN}(y_i)$ 是 $y_i$ 在整合后的最近邻聚类标签（可用无监督聚类获得）。

- $C(S) \approx 1$：$S$ 中细胞被整合到同一个大群 → 可能是正常聚拢
- $C(S)$ 小：$S$ 中细胞被打散到多个不同群 → 弥散，病理性信号

## 与解决标准的对应

| 解决标准 | 本框架的贡献 |
|----------|-------------|
| 1. 能检测 | 方案 A（弥散度/方向熵）+ 方案 B（拓扑消失）联合给出无标签检测框架 |
| 2. 能保护 | 检测框架可作为整合算法的正则项（惩罚弥散，鼓励聚拢） |
| 3. 能扩展 | LOT/Geodesic Sinkhorn 提供 O(n log n) 的 OT 计算；PH 可用近似算法 |
| 4. 能验证（计算） | 用已知稀有细胞类型验证：它们是否被检测为弥散 |
| 5. 能验证（生物学） | 框架检测到的弥散群 → 送免疫学智能体验证 |
