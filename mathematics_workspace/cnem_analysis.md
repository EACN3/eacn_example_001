# 数学智能体 — CNEM 指标的数学理论分析

## 1. CNEM 的形式化定义

设细胞 $i$ 的原始归一化表达向量为 $x_i \in \mathbb{R}^G$（$G$ 为基因数），整合后嵌入为 $z_i \in \mathbb{R}^d$。

整合后 $k$-近邻集合 $N_{\text{post}}(i) = \{j_1, \ldots, j_k\}$（在 $z$ 空间中的 $k$-NN）。

$$\text{CNEM}(i) = 1 - \cos(x_i, \bar{x}_{N(i)})$$

其中 $\bar{x}_{N(i)} = \frac{1}{k}\sum_{j \in N_{\text{post}}(i)} x_j$ 为邻域平均表达。

## 2. 为什么 post_CNEM 对稀有亚群有效？

### 2.1 几何直觉

在高维表达空间 $\mathbb{R}^G$ 中，每个细胞类型占据一个"锥"（cone）——同类细胞的表达向量方向相近。余弦相似度恰好度量的是锥角距离。

**正常（大群）细胞**：整合后，邻居仍以同类细胞为主 → $\bar{x}_{N(i)}$ 与 $x_i$ 方向一致 → $\text{CNEM}(i)$ 小。

**稀有亚群细胞（被弥散后）**：整合后被推入大群，邻居变成不同类型 → $\bar{x}_{N(i)}$ 方向偏向大群的表达模式 → 与 $x_i$ 的夹角变大 → $\text{CNEM}(i)$ 大。

### 2.2 形式化证明

**定理（CNEM 的分离性）**：设存在两种细胞类型 $A$（大群）和 $B$（稀有群），其表达均值分别为 $\mu_A, \mu_B \in \mathbb{R}^G$，且 $\cos(\mu_A, \mu_B) = \rho < 1$。若整合后稀有群细胞 $i \in B$ 的 $k$-近邻中有比例 $\alpha$ 来自大群 $A$，则：

$$\mathbb{E}[\text{CNEM}(i)] \geq 1 - \frac{1}{\sqrt{1 + \alpha^2(1-\rho^2)\|\mu_A\|^2/\sigma_B^2}}$$

其中 $\sigma_B^2$ 是 $B$ 群内方差。

**推论**：当 $\alpha \to 1$（稀有群被完全弥散到大群中）且 $\rho$ 远小于 1（两种类型表达差异大）时，CNEM 显著升高。

### 2.3 为什么 delta 失败而绝对值成功？

**关键洞察**：稀有亚群在整合前就处于表达空间的边缘位置。

- **整合前**：稀有群细胞的邻居多来自批次内同类细胞（因为嵌入空间未对齐）。但表达空间中，跨批次的邻居可能已经是异类 → CNEM_pre 可能已经很高。
- **整合后**：稀有群被推入大群，嵌入空间的邻居变为大群细胞 → CNEM_post 很高。
- **Delta = post - pre**：如果 pre 已经高（稀有群本身就 mismatch），post 也高，delta 可能很小甚至为负。

数学上：$\text{CNEM}_{\text{pre}}$ 的高低取决于整合前嵌入空间中的邻居组成，这受批次效应影响很大，不是一个稳定的参考基线。而 $\text{CNEM}_{\text{post}}$ 直接反映整合后的几何关系——"你在嵌入空间中被放在了谁旁边，你跟他们在表达空间中像不像？"——这是一个更干净的信号。

## 3. 理论下界与失效条件

### 3.1 理论下界

对任意细胞 $i$：

$$\text{CNEM}(i) \geq 0$$

等号成立当且仅当 $x_i$ 与 $\bar{x}_{N(i)}$ 方向完全一致（完美匹配）。

**非平凡下界**：设 $x_i$ 属于类型 $B$，邻域中类型 $A$ 的比例为 $\alpha$。在高斯假设下：

$$\text{CNEM}(i) \gtrsim \frac{\alpha^2 \sin^2\theta_{AB}}{2}$$

其中 $\theta_{AB} = \arccos(\rho)$ 是两种类型均值间的锥角。

### 3.2 失效条件

CNEM 在以下情况下会失效（无法区分稀有群与大群）：

**(F1) 表达相似的稀有亚群**：若 $\rho = \cos(\mu_A, \mu_B) \approx 1$（稀有群与大群的表达模式几乎相同），则即使稀有群被弥散，CNEM 仍接近 0。这对应于仅靠少量基因区分的亚型（如同一分化谱系中的相邻阶段）。

**(F2) 整合不充分**：若整合算法未能有效对齐批次，大群细胞的邻居中也混入了异类细胞，导致大群的 CNEM 也偏高 → 信噪比下降。

**(F3) k 选取不当**：若 $k$ 远大于稀有群细胞数 $n_B$，则稀有群被"稀释"得过于彻底，但大群也可能混入远处的异类邻居。若 $k$ 过小，估计不稳定。最佳选择：$k \sim \sqrt{N}$ 或基于邻域稳定性自适应选取。

**(F4) 基因空间维度灾难**：当有效基因维度 $G_{\text{eff}}$ 很高时，高维空间中余弦相似度趋向于集中在某个窄范围内（concentration of measure），降低区分力。

## 4. 改进建议

### 4.1 加权 CNEM (wCNEM)

当前 CNEM 对所有近邻等权平均，但更近的邻居应该更有信息量。定义：

$$\text{wCNEM}(i) = 1 - \cos\left(x_i, \frac{\sum_{j \in N(i)} w_{ij} x_j}{\sum_{j \in N(i)} w_{ij}}\right)$$

其中 $w_{ij} = \exp(-\|z_i - z_j\|^2 / (2h^2))$（高斯核），$h$ 为带宽参数。

**优势**：减少远距离邻居的噪声贡献，提高局部灵敏度。

### 4.2 核化 CNEM (kCNEM)

将余弦相似度替换为更一般的核函数：

$$\text{kCNEM}(i) = 1 - \frac{1}{k}\sum_{j \in N(i)} K(x_i, x_j)$$

其中 $K$ 可以是：
- **RBF 核**：$K(x,y) = \exp(-\|x-y\|^2/(2\sigma^2))$ — 对欧氏距离敏感
- **多项式核**：$K(x,y) = (\langle x,y \rangle + c)^p$ — 捕捉非线性结构

这本质上变成了对每个细胞在其邻域上计算局部 MMD（Maximum Mean Discrepancy）的单侧版本。

### 4.3 自适应 k-选取

用局部密度自适应选取 k 值：

$$k(i) = \min\{k : \text{局部密度估计的变异系数 CV}(i) < \tau\}$$

这使得稀疏区域用较大的 k（降低方差），密集区域用较小的 k（提高分辨率）。

### 4.4 多尺度 CNEM

在多个 $k$ 值上计算 CNEM 并取加权平均或最大值：

$$\text{msCNEM}(i) = \max_{k \in \{k_1, \ldots, k_m\}} \text{CNEM}_k(i)$$

**理论动机**：不同大小的稀有群在不同尺度上信号最强。多尺度方法自动适配。

## 5. 与 LOF / Isolation Forest 的比较

| 维度 | CNEM | LOF | Isolation Forest |
|------|------|-----|-----------------|
| **检测空间** | 跨空间：嵌入空间定邻居，表达空间度量异常 | 单一空间中的密度 | 单一空间中的可分离性 |
| **核心信号** | 表达 mismatch（语义层面） | 密度比（几何层面） | 分割深度（统计层面） |
| **对稀有群的灵敏度** | 高：稀有群被弥散后必然与邻居不匹配 | 中：稀有群如果被推入大群密集区，密度反而正常 | 中：类似问题 |
| **利用整合信息** | 是：专门比较整合前后的空间差异 | 否：只看单一空间 | 否 |
| **假阳性控制** | 好：仅当表达真正不同时才报警 | 差：低密度区域的正常细胞也被标记 | 中等 |
| **计算复杂度** | O(Nk + Nk·G) 含 KNN | O(Nk·log N) | O(N·T·log N) |
| **关键优势** | 利用了"嵌入空间-表达空间不一致"这一整合病理的核心特征 | 不需要两个空间 | 对高维数据鲁棒 |
| **关键劣势** | 依赖 k 选取和嵌入质量 | 不能区分批次效应导致的低密度和生物学上的稀有 | 无法利用整合特有的信息 |

**核心理论优势**：CNEM 利用了整合产生的独特信息结构——嵌入空间和表达空间的不一致性。LOF 和 Isolation Forest 只在单一空间中操作，无法利用这种跨空间的诊断信号。这正是 CNEM 对整合后稀有亚群检测更有效的根本数学原因。
