# 稀有亚群信号保留率：端到端理论分析

## 乘法压缩模型

设稀有亚群占比 f=0.5%，原始信号强度 S₀=1。各环节的信号保留率相乘。

## 当前标准流程

| 环节 | 方法 | 保留率 | 累积信号 |
|------|------|--------|---------|
| 原始 | — | 1.00 | 1.000 |
| HVG | 全局方差Top2000 | 0.30 | 0.300 |
| PCA | 全局方差Top50PC | 0.01 | 0.003 |
| 整合 | Harmony (OCF=5.6) | 0.20 | 0.0006 |
| **总保留率** | | | **0.06%** |

### 各环节压缩机制

**HVG（保留30%）**：稀有亚群的 marker 基因仅在 0.5% 细胞中高表达，全局方差低。约 70% 的稀有 marker 被排除在 HVG 之外。

**PCA（保留1%）**：PCA 最大化全局方差。稀有亚群对协方差矩阵 Σ 的贡献为 f·Σ_rare ≈ 0.005·Σ_rare。前50个PC由大群主导，稀有亚群特有方向被压缩到第50+个PC中。

**Harmony（保留20%）**：OCF=5.6 意味着校正幅度是实际需要的5.6倍。稀有亚群在PCA空间中已经微弱的信号被进一步覆盖。

## 修复后流程

| 环节 | 方法 | 保留率 | 累积信号 |
|------|------|--------|---------|
| 原始 | — | 1.00 | 1.000 |
| HVG | 密度反比加权 | 0.70 | 0.700 |
| PCA | 1/density加权PCA | 0.40 | 0.280 |
| 整合 | 选择性整合(OCF≈1) | 0.90 | 0.252 |
| **总保留率** | | | **25.2%** |

### 各环节修复机制

**加权HVG（保留70%）**：对低密度区域的细胞赋予更高权重计算方差，稀有 marker 的加权方差提升，更多被保留。

**加权PCA（保留40%）**：对协方差矩阵加密度反比权重 Σ_w = Σ (1/ρ_i)·x_i·x_i^T。稀有亚群的贡献从 f·Σ_rare 提升到 ~Σ_rare（与大群等权），其特有方向进入前50 PC。

**选择性整合（保留90%）**：OCF≈1，仅校正共享细胞。稀有亚群因无跨批次对应而不被校正，信号几乎完整保留。10%损失来自边缘效应。

## 提升倍数分析

| 修复 | 单独增益 | 累积增益 |
|------|---------|---------|
| HVG→加权HVG | 2.3x | 2.3x |
| PCA→加权PCA | **40x** | 93x |
| Harmony→SI | 5x | **417x** |

## 核心结论

1. **总提升 ~400 倍**：从 0.06% 到 25%
2. **PCA 是最大瓶颈**：单步修复贡献 40x（占总增益的 ~70%）
3. **任何单独修复不够**：仅修复整合（5x）仍只有 0.3%；仅修复 PCA（40x）仍只有 2.4%
4. **三环节同时修复是必要的**——支持"端到端稀有性感知流程"的论文定位
