## 核心叙事：从范式盲区到范式转换

### 第一幕：问题的发现（Introduction）

**叙事线**：不是"现有方法不够好"，而是"现有范式存在结构性盲区"。

- 所有现有评估框架都预设"需要保护的类型已在标签中"
- 这使得未知稀有亚群的消失**在概念上不可观察**——不是检测不到，而是根本没有被定义为需要检测的对象
- 这是一个 Kuhnian 范式盲区：框架的隐含假设排除了某些现象被观察到的可能性

**建议开篇策略**：不要从"批次效应很重要"开始（审稿人已经知道了），而是从一个具体案例开始——"在X数据集中，Y亚群在整合后完全消失，但没有任何现有指标的值发生变化"。用一个生动的失败案例打开问题。

### 第二幕：为什么至今没有解决（Background/Analysis）

**叙事线**：三重缺失的自封闭循环。

- 没有 ground truth → 没有评估框架 → 没有失败信号 → 没有算法开发动力
- 不是因为问题不重要，而是因为循环入口被锁死
- 所有人默认 ground truth 必须从外部获取——**这个默认假设本身就是问题的一部分**

### 第三幕：突破口（Key Insight）

**叙事线**：信号存在于数据本身。

- 正常批次校正（聚拢）和稀有亚群消灭（弥散）在数据结构上模式不同
- 不需要外部 ground truth，可以从整合前后的结构变化本身区分
- 这是一次概念转换：从"监测已知类型"到"监测结构变化模式"

### 第四幕：方法与挫折（Methods — 包含CNEM失败）

**叙事线**：坦诚的探索过程。

建议采用"探索式叙事"而非"成功式叙事"：

1. **首先尝试CNEM**（表达-邻域不匹配）：在胰腺数据上有效(AUC 0.76-0.80)
2. **CNEM在免疫数据上失败**(AUC 0.37-0.40)：揭示了一个深层问题——基于表达距离的检测在转录组重叠大的数据中失效
3. **关键洞察**：不存在universal detector，检测器有效性与数据内在结构耦合
4. **路径转向**：从表达空间转向拓扑/结构空间（DS弥散分数等）
5. **最终框架**：自适应检测策略

**为什么包含失败更好**：
- 审稿人更信任"经过严格测试并坦诚报告失败"的工作
- CNEM失败本身就是一个科学发现——揭示了检测器-数据耦合效应
- 展示了从失败到洞察到更好方案的科学推理过程
- Popper式叙事：方法通过了哪些证伪尝试，而非被验证了多少次

### 第五幕：验证（Results）

**叙事线**：多层递进验证。

1. **计算验证（已知亚群）**：用pDC等已知稀有亚群，隐藏标签，检验框架能否检出
2. **留出验证**：开发集(pDC) vs 验证集(Mast/T cell亚群)
3. **盲测验证**：框架检测到的"被消灭的未知亚群"——事后用marker基因确认身份
4. **湿实验验证**：框架预测的候选亚群通过实验证实存在

### 第六幕：意义（Discussion）

**叙事线**：这不仅是一个方法，而是一次认识论转换。

- 从"已知驱动"到"结构驱动"的评估范式
- 对所有使用批次整合的研究的启示：你的整合结果中可能缺失了什么？
- 预防原则：在不确定某个结构是否有意义时，默认保护
- 局限性的坦诚讨论：吸收模式的检测困难、最小可检测群大小、可扩展性

## 标题建议

弱标题（描述方法）：
- "A framework for detecting rare subpopulation loss in single-cell batch integration"

强标题（传达范式转换）：
- "Detecting the undetectable: label-free identification of rare subpopulation loss in single-cell batch integration"
- "Beyond known labels: structural change detection reveals hidden casualties of batch integration"
- "The blind spot of batch integration: an unsupervised framework for protecting unknown rare subpopulations"

推荐第三个——"blind spot"直接传达范式盲区的核心信息。

## 审稿人可能的挑战与预备回应

| 挑战 | 预备回应 |
|------|---------|
| "你们检测到的小群可能只是噪声" | 四条本体论标准（跨批次一致性、属性聚簇稳定性、功能可预测性、可区分性） |
| "为什么不直接用已有注释评估？" | 这正是我们要打破的循环——已有注释覆盖不了未知亚群 |
| "方法在其他数据上也有效吗？" | 胰腺(有效) + 免疫(调整后有效) + 不同规模的留出验证 |
| "CNEM失败说明方法不可靠" | 失败揭示了检测器-数据耦合效应，因此我们发展了自适应策略 |
| "样本量不够" | 225万细胞、103批次、8种细胞类型——真实世界规模 |
| "湿实验只验证了一个亚群" | 计算验证覆盖了多个已知亚群，湿实验是最终的实在论确认 |
