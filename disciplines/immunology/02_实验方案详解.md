# 02 — 实验方案详解

## 一、细胞来源与分选

### 1.1 外周血单个核细胞（PBMC）分离

- **来源**：健康供体外周血
- **方法**：Ficoll-Paque PLUS（GE Healthcare）密度梯度离心
- **原理**：血液分层后，PBMC（淋巴细胞、单核细胞）集中在血浆和 Ficoll 的界面层，红细胞和粒细胞沉到底部

### 1.2 TIGIT⁺ Treg 的流式分选

PBMC 用以下 5 种荧光抗体进行表面染色：

| 抗体靶点 | 荧光素 | 克隆号 | 厂商 | 用途 |
|---|---|---|---|---|
| CD4 | APC | RPA-T4 | BioLegend | 标记 CD4⁺ T 细胞 |
| CD25 | PE/Cy7 | BC96 | BioLegend | 标记活化/调节性 T 细胞（IL2RA） |
| CD127 | BV421 | A019D5 | BioLegend | Treg 低表达 CD127，用于排除非 Treg |
| TIGIT | PE | A15153G | BioLegend | 标记 TIGIT⁺ 亚群 |
| CCR8 | BV711 | L263G8 | BioLegend | 排除 CCR8⁺ 亚群 |

**分选策略（Gating）**：

```
全部 PBMC
  └→ CD4⁺（选出 CD4 T 细胞）
      └→ CD25⁺ CD127⁻/lo（选出 Treg）
          └→ TIGIT⁺ CCR8⁻（选出目标亚群）
```

- **设备**：BD FACSAria II（BD Biosciences）
- **最终得到**：CD4⁺CD25⁺CD127⁻/lo TIGIT⁺CCR8⁻ 的 Treg 亚群

### 1.3 CD8⁺ 效应 T 细胞分离

- **方法**：阴性选择（磁珠分选）
- **试剂盒**：EasySep Human CD8⁺ T Cell Isolation Kit（STEMCELL Technologies）
- **原理**：用抗体标记所有"非 CD8⁺ T 细胞"的细胞，磁珠吸附带走，剩下的就是纯的 CD8⁺ T 细胞

## 二、共培养实验

### 2.1 实验条件

| 参数 | 设置 |
|---|---|
| 共培养比例 | Treg : 效应 T 细胞 = 1 : 4 |
| 刺激条件 | anti-CD3/CD28 Dynabeads（Thermo Fisher）模拟抗原刺激 |
| 培养时间 | 72 小时 |
| 培养基 | RPMI-1640 完全培养基 |
| 血清 | 10% FBS |
| 抗生素 | 100 U/mL 青霉素-链霉素 |
| 细胞因子 | 100 IU/mL 重组人 IL-2（PeproTech） |

### 2.2 实验分组

| 组别 | 处理 |
|---|---|
| NC（阴性对照） | CD8⁺ T 细胞 + Dynabeads，不加 Treg |
| TIGIT⁺ Treg 组 | CD8⁺ T 细胞 + Dynabeads + TIGIT⁺ Treg（1:4） |

### 2.3 为什么用 anti-CD3/CD28 Dynabeads？

体外实验中 T 细胞不会自己活化，需要模拟体内抗原呈递细胞（APC）提供的两个信号：
- **信号 1**：anti-CD3 模拟 TCR 识别抗原的刺激
- **信号 2**：anti-CD28 提供共刺激信号

Dynabeads 是包被了这两种抗体的磁珠，相当于"人工 APC"。

## 三、三个检测实验

### 实验 1：T 细胞活化检测（Figure 5G）

| 项目 | 详情 |
|---|---|
| 检测指标 | CD69⁺ T 细胞比例 |
| 检测方法 | 流式细胞术 |
| 染色抗体 | anti-CD69-FITC（克隆 FN50，BioLegend） |
| 检测设备 | CytoFLEX（Beckman Coulter） |
| CD69 的意义 | T 细胞活化最早期的表面标志物之一，活化后数小时内上调 |

**逻辑**：如果 TIGIT⁺ Treg 抑制了 T 细胞活化，CD69⁺ T 细胞的比例应该下降。

### 实验 2：IFN-γ 分泌检测（Figure 5H）

| 项目 | 详情 |
|---|---|
| 检测指标 | 培养上清中 IFN-γ 浓度（pg/mL） |
| 检测方法 | ELISA（酶联免疫吸附） |
| 试剂盒 | Human IFN-γ Quantikine ELISA Kit（R&D Systems，货号 DIF50C） |
| IFN-γ 的意义 | CD8⁺ T 细胞杀伤肿瘤的关键效应因子，促进肿瘤细胞凋亡和免疫招募 |

**逻辑**：如果 TIGIT⁺ Treg 削弱了 T 细胞的效应功能，IFN-γ 分泌应该减少。

### 实验 3：肿瘤杀伤实验（Figure 5I）

| 项目 | 详情 |
|---|---|
| 检测指标 | 肿瘤细胞杀伤率（%） |
| 检测方法 | LDH 释放实验 |
| 试剂盒 | CytoTox 96 Non-Radioactive Cytotoxicity Assay（Promega，货号 G1780） |
| 靶细胞 | A549 人非小细胞肺癌细胞系 |
| 效靶比 | 10 : 1（效应 T 细胞 : 肿瘤细胞） |
| 杀伤时间 | 24 小时 |

**实验流程**：

```
第一阶段（72h）：                    第二阶段（24h）：
┌──────────────────────┐           ┌──────────────────────┐
│ CD8⁺ T 细胞           │           │ 取出 T 细胞           │
│   ± TIGIT⁺ Treg      │  ──72h→  │   + A549 肿瘤细胞     │  ──24h→  测 LDH
│   + Dynabeads        │           │   （10:1）            │
└──────────────────────┘           └──────────────────────┘
```

**LDH 释放实验原理**：
- 细胞死亡时，细胞膜破裂，胞内的乳酸脱氢酶（LDH）释放到培养液中
- 检测上清中 LDH 活性即可定量细胞死亡程度

**杀伤率计算公式**：

```
杀伤率(%) = (实验组 - 效应细胞自发释放 - 靶细胞自发释放) / (靶细胞最大释放 - 靶细胞自发释放) × 100
```

其中：
- 实验组 = T 细胞 + 肿瘤细胞共培养的 LDH
- 效应细胞自发释放 = T 细胞单独培养的 LDH（背景噪声）
- 靶细胞自发释放 = 肿瘤细胞单独培养的 LDH（自然死亡）
- 靶细胞最大释放 = 肿瘤细胞裂解液的 LDH（100% 死亡时的 LDH 量）
