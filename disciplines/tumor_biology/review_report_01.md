# 肿瘤生物学审视报告：单细胞批次整合中稀有亚群保护框架

## 任务背景

团队正在开发检测单细胞批次整合中未知稀有亚群丢失的框架。核心指标是 CNEM（Cell-Neighborhood Expression Mismatch）。数据科学发现 pDC 内部存在26个亚群，其中极稀有亚群（如 cluster 24，93细胞，仅NHL）集中在特定癌种。

---

## Q1：最可能在批次整合中丢失的稀有免疫亚群

| 亚群 | 为何脆弱 | 临床意义 |
|------|---------|---------|
| **耗竭前体T细胞 (Tpex / TCF1+PD-1+ CD8 T)** | 占肿瘤浸润CD8 T细胞的1-5%，具有独特的TCF7/BCL6表达谱，与终末耗竭T细胞和效应T细胞在转录组上有重叠但又有关键差异。整合时极易被推入更大的耗竭T细胞群。 | 免疫检查点治疗（anti-PD-1）的主要响应细胞群，丢失意味着丧失预测免疫治疗反应的能力。 |
| **cDC3 / AS-DC (AXL+SIGLEC6+ 树突状细胞)** | 极稀有（<0.5%免疫细胞），转录特征介于cDC2和pDC之间，整合时极易被吸收进任一大群。 | 可能在抗肿瘤抗原呈递中扮演独特角色，其丢失会影响DC-T细胞互作分析。 |
| **TIGIT+ Treg 亚群** | Treg本身占比不高，TIGIT+亚型更是其中的子集。不同癌种中比例差异大，跨批次整合时易被主流Treg群吞没。 | TIGIT+Treg是肿瘤微环境中免疫抑制的关键执行者，与anti-TIGIT联合免疫治疗密切相关。 |
| **肿瘤相关肥大细胞** | 在多数实体瘤中占免疫浸润<1%，且测序捕获效率低（颗粒含量高导致RNA质量差异）。 | 在肿瘤血管生成和免疫调节中有双重作用，是被系统性忽视的免疫亚群。 |
| **三级淋巴结构(TLS)中的滤泡辅助T细胞 (Tfh)** | 仅存在于形成TLS的肿瘤样本中，批次来源高度不均，整合时跨批次分布极度偏斜。 | TLS是免疫治疗预后的重要标志物，Tfh是TLS功能的核心。 |
| **肿瘤干细胞 / 癌症起始细胞 (CSC)** | 极稀有（通常<1%肿瘤细胞），且部分转录特征与正常干细胞或增殖细胞重叠。 | 治疗耐药和复发的根源，丢失意味着错过最关键的治疗靶点。 |

## Q2：pDC 在肿瘤中的功能异质性

| 功能状态 | 标志物 | 功能 |
|---------|--------|------|
| **经典IFN-I产生型 pDC** | IRF7high, TLR7/9high, IFNAhigh | 产生大量I型干扰素，具有抗肿瘤活性。在多数肿瘤中被抑制。 |
| **耐受性/免疫抑制型 pDC** | IDOhigh, ICOS-Lhigh, IFNAlow | 诱导Treg分化，促进免疫耐受。肿瘤微环境中的主导状态。 |
| **抗原呈递活化型 pDC (AXL+SIGLEC6+)** | AXLhigh, SIGLEC6high, CD86high | 介于pDC和cDC之间的过渡状态，可能具有独特的抗原呈递能力。即cDC3/AS-DC。 |
| **癌种特异性pDC亚群** | 待定，需结合具体数据 | 如cluster 24（93细胞，仅NHL）——NHL(非霍奇金淋巴瘤)中pDC可能存在特有的功能状态，与B细胞淋巴瘤的独特微环境有关。pDC通过TLR信号在B细胞淋巴瘤发生中有已知作用。 |

**对CNEM的启示**：pDC的这种异质性意味着整合算法很可能将功能不同的pDC亚群合并。CNEM指标如果能捕捉到整合前后pDC内部结构的"弥散"，将非常有价值。

## Q3：TIGIT+ Tregs 与检测到的稀有亚群的关联

- TIGIT+Tregs是肿瘤微环境中功能最强的免疫抑制性Treg亚群，通过TIGIT-CD155轴抑制NK和CD8 T细胞功能。
- 如果湿实验数据中包含TIGIT+Tregs的表征（如流式细胞术TIGIT/FOXP3共染），这为计算框架提供了理想的交叉验证机会。
- 关键假设：如果CNEM框架检测到某个被整合消灭的Treg亚群，且该亚群的marker基因中包含TIGIT及其下游通路基因（如FOXP3, IL10, CTLA4, IL2RA），则湿实验数据可直接验证其存在性。
- 癌种特异性：TIGIT+Tregs在黑色素瘤、NSCLC、NHL中比例差异大，这正是跨批次整合时容易丢失的特征——某些批次（癌种）有、另一些没有。

## Q4：最值得湿实验验证的生物学特征

**优先级排序（全局仅一次湿实验机会）：**

1. **具有可操作的表面标志物** — 湿实验（流式、FACS分选）需要表面marker。如果该亚群的差异基因中包含可用抗体的表面蛋白（如CD/TIGIT/ICOS/AXL等），验证可行性最高。
2. **癌种特异性分布** — 如果该亚群集中在1-2个癌种中（如cluster 24仅在NHL），则验证时只需从特定癌种取样，实验设计更聚焦。
3. **与已知功能通路有可解释的关联** — 差异基因应可映射到已知生物学通路（如IFN信号、T细胞耗竭、免疫检查点等），这样即使是新亚群，也能构建功能假说。
4. **与免疫治疗响应或预后相关** — 如果该亚群的marker基因与已知的免疫治疗biomarker重叠（如PD-L1通路、TLS标志物），其临床意义最容易论证，Nature级论文的影响力最大。

**具体建议**：最理想的验证目标是一个位于pDC或Treg谱系内、具有表面marker、仅存在于特定癌种中、且与免疫检查点通路相关的未知亚群。
