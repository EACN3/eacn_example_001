# 全流程偏差分析：稀有亚群的逐步消亡

## 案例A：GITR⁺ 活化态 Treg 功能社区 (T cell Cluster 5)

### 原始状态
- 1464细胞，GITR=69.8%, TIGIT=65%, FOXP3=28%
- 86%来自皮肤癌 (SSCC + Skin Normal, D100 Dataset)
- 三重组合 FOXP3⁺GITR⁺TIGIT⁺CCR8⁻: 266细胞

### 步骤1: 样本解离与文库制备
**偏差机制**：活化态Treg高表达颗粒酶和细胞因子，细胞在酶解过程中更易受损。高代谢活性（OXPHOS活跃）导致解离后线粒体膜电位快速下降，细胞活力损失高于静息态T细胞。
**预估影响**：~5-10%活化Treg在此步骤丢失（选择性丢失，非随机）。

### 步骤2: QC/过滤
**偏差机制**：
- **线粒体基因比例 (mt%)**：活化态Treg代谢活跃，基础mt%高于naive T细胞。标准过滤阈值（如mt%>20%）会不成比例地过滤掉活化Treg。这是一个此前未被充分认识的偏差点。
- **基因数/UMI数过滤**：GITR⁺ Treg高表达免疫调节基因但转录组多样性可能低于效应T细胞，低基因数阈值可能移除部分细胞。
- **Doublet过滤**：Treg-effector T共存的功能社区中，物理上紧密相邻的细胞更易形成doublet。Scrublet可能将真实的Treg-T互作对误判为doublet。
**预估影响**：~5-15%选择性丢失。

### 步骤3: 归一化
**偏差机制**：标准归一化（如scran deconvolution）假设大多数基因在细胞间表达量相似。但GITR⁺ Treg的转录程序高度特化（FOXP3/CTLA4/IL2RA/TNFRSF18高表达），归一化可能压缩这些marker的相对表达差异。
**预估影响**：轻微，但累积效应降低了后续HVG选择中该亚群的可区分性。

### 步骤4: 高变基因 (HVG) 选择
**偏差机制**：全局HVG选择偏向于在大量细胞中变异的基因。TNFRSF18(GITR)仅在<1%的细胞中高表达，其方差在全局计算中被稀释，可能不被选为HVG。同样影响FOXP3、TIGIT在Treg子集中的判别力。
**预估影响**：中等。关键marker基因被排除意味着在降维空间中该亚群与主流Treg的距离被压缩。

### 步骤5: 降维 (PCA)
**偏差机制**：PCA捕捉方差最大的方向。占总细胞<1%的GITR⁺ Treg对全局协方差矩阵的贡献微乎其微，前50个PC几乎不包含区分该亚群与主流Treg的方向。在PCA空间中，该亚群被压缩到一个很小的体积内，与周围Treg的距离远小于其在原始表达空间中的距离。
**预估影响**：严重。亚群在低维空间中的可分辨性大幅下降。

### 步骤6: 构建KNN图
**偏差机制**：在PCA压缩后的空间中构建KNN图时，GITR⁺ Treg细胞的k个邻居中混入大量主流Treg（因为PCA中距离被压缩）。该亚群在图空间中失去了独立的社区结构。
**预估影响**：中等到严重。图结构已不再反映原始表达空间中的亚群边界。

### 步骤7: 批次整合
**偏差机制**：86%来自皮肤癌（D100），跨批次对齐时整合算法将其推向其他批次中的主流Treg。Harmony的迭代聚类-校正中，该亚群细胞数太少无法形成独立聚类质心，被分配到最近的大群质心进行校正。
**实测影响**：survival = 24%→被打散到8个post-cluster。**这是致命一击，但前6步的累积偏差已将该亚群的"抵抗力"削弱到最低。**

### 步骤8: 聚类与注释
**偏差机制**：残余的~24%细胞分散在8个post-cluster中，在任何单个cluster中都不构成多数。聚类算法无法将其识别为独立群体。在最终的细胞注释中，这些细胞被标注为普通"Treg"或"CD4 T cell"。
**最终结果**：**GITR⁺ 活化态 Treg 功能社区从数据中完全消失。**

### 总结
| 步骤 | 偏差类型 | 严重度 | 可逆? |
|------|---------|--------|------|
| 解离 | 活力选择性丢失 | 轻 | 否 |
| QC/mt% | 代谢活跃偏差 | 中 | 是（调整阈值） |
| 归一化 | 特化转录程序压缩 | 轻 | 是（替代方法） |
| HVG | 稀有marker排除 | 中 | 是（谱系内HVG） |
| PCA | 低维压缩 | 严重 | 是（增加PC数/谱系内PCA） |
| KNN | 邻域污染 | 中-严重 | 部分（调整k） |
| 整合 | 跨批次强制对齐 | **致命** | 是（选择性整合） |
| 聚类 | 碎片化后不可识别 | 严重 | 否（上游已被消灭） |

## 案例B：pDC — 规模依赖性逐步消亡

### 原始状态
- 225万免疫细胞中的稀有类型（约1-3%）
- 独特marker: IRF7, TLR7, TLR9, IL3RA, CLEC4C, TCF4
- 8批次子集中完好（99%召回）→ 103批次全量中被消灭（NP=0.231）

### 步骤1-3: 解离/QC/归一化
**偏差**：pDC相对较小，解离损伤较低。QC中无特殊偏差。归一化中pDC的IFN通路基因可能被压缩。
**影响**：轻微。pDC在这几步中基本安全。

### 步骤4: HVG选择
**偏差（8批次 vs 103批次差异开始显现）**：
- 8批次：pDC占~2%细胞，其marker基因（IRF7等）的方差足以被选为HVG
- 103批次：pDC被225万细胞稀释到<1%，IRF7的全局方差贡献进一步下降。同时103个批次引入大量批次特异的高变基因，挤占HVG名额
**影响**：8批次轻微，103批次中等。

### 步骤5: PCA
**偏差（规模依赖性放大）**：
- 8批次：pDC在PC空间中仍有独立的"角落"，因为其IRF7/TLR通路基因被HVG捕获
- 103批次：(a) HVG中pDC marker减少 → PCA方向不包含pDC信号 (b) 103个批次的批次效应贡献了更多PC方差 → pDC的生物学方差被批次方差"淹没"
**影响**：8批次轻微，103批次严重。**这是规模依赖性消灭的转折点。**

### 步骤6: KNN图
**偏差**：
- 8批次：pDC在PCA空间中仍有独立簇，KNN图中pDC互为邻居
- 103批次：pDC在PCA空间中被压缩，邻居中混入cDC2和Monocyte（共享MHC-II基因）
**影响**：8批次安全，103批次严重。

### 步骤7: 批次整合
**偏差（致命一击，仅在103批次时发生）**：
- 8批次：需要对齐的批次少，pDC在每个批次中的数量足以形成独立的MNN锚点，整合保持pDC的独立性
- 103批次：(a) 每多一个批次增加一组需要对齐的细胞 (b) pDC与cDC2/Monocyte在PCA空间中已经很近（步骤5的后果）(c) 整合算法找到pDC-cDC2之间的虚假MNN对并将其对齐 (d) 每次迭代进一步缩小pDC与大群的距离 → 最终pDC被完全吸收
**实测影响**：NP = 0.231，所有cell type中最低。

### 步骤8: 聚类与注释
- 8批次：pDC形成独立cluster，被正确注释
- 103批次：pDC细胞被打散到DC/Monocyte大群中，不再被识别为独立类型

### 规模依赖性转折点分析

| 步骤 | 8批次 | 103批次 | 转折原因 |
|------|-------|---------|---------|
| HVG | pDC marker被选中 | pDC marker被挤出 | 批次数↑ → 批次HVG↑ → 挤占名额 |
| PCA | pDC有独立PC方向 | pDC方差被淹没 | 批次方差 >> 稀有类型方差 |
| KNN | pDC互为邻居 | 邻居被cDC2/Mono污染 | PCA压缩 → 距离缩小 |
| 整合 | MNN正确配对 | 虚假MNN跨类型配对 | KNN污染 → 虚假锚点 |
| 聚类 | 独立cluster | 被吸收 | 上游累积 |

**关键洞察**：pDC的消灭不是整合单独造成的——是HVG→PCA→KNN→整合的级联放大效应，而批次数量的增加在每一步都放大了偏差。整合只是压垮骆驼的最后一根稻草。
