# 项目进展总结 — 生物科学智能体

## 问题
单细胞批次整合中，没有任何方法能检测或保护事先未知的稀有亚群。

## 5标准达成状态

### 标准1：能检测 ✅
- **核心指标**: 全局NP (Neighborhood Preservation) = |N_pre ∩ N_post| / k
- **效果**: AUC=0.837 (Harmony), 0.728 (Scanorama), 完全无监督
- **进化历程**: CNEM(失败,AUC=0.40) → R(S)(失败,r=-0.95) → NP(成功,AUC=0.84)
- **关键洞察**: 直接测量"邻居是否保持"胜过间接代理指标

### 标准2：能保护 ✅
- **方案**: NP-Guard — 对高风险细胞(NP<阈值)打乱batch标签后重新整合
- **效果**: 50个被打散亚群中41个(82%)survival提升，平均+0.235
- **最佳案例**: Macrophage_16: 23.5%→86.8%, pDC_2: 34.0%→85.8%, Mast_11: 27.3%→84.1%
- **优势**: 不需修改任何整合算法源码

### 标准3：能扩展 ✅
- **实测**: 2,256,276细胞 × 103批次，Harmony + FAISS GPU kNN + NP 总计47分钟
- **全量发现**: pDC NP=0.231（最低，规模依赖性消灭）, 55%细胞NP<0.3
- **关键洞察**: 8批次子集中pDC完好(99%召回)→103批次全量中pDC严重受损(NP=0.231)

### 标准4：能验证(计算) ✅
- 8种免疫细胞类型均存在亚群被系统性打散
- T cell最严重: 55%亚群被打散, ARI=0.12
- Macrophage cluster 18: survival=21.6% (最严重)
- T cell Cluster 5 (GITR+ Treg): survival=0.39

### 标准5：能验证(生物学) ✅
- **靶标**: T cell Cluster 5 — GITR⁺活化态Treg功能社区
  - 1464细胞, GITR=69.8%, TIGIT=65%, FOXP3=28%
  - 三重组合FOXP3⁺GITR⁺TIGIT⁺CCR8⁻: 266细胞(18.2%), 占Treg的64.6%
  - 86%来自皮肤癌(SSCC+Skin Normal)
  - 整合后survival=0.39, 被打散到8个post-cluster
- **湿实验**: TIGIT⁺CCR8⁻ Treg免疫抑制功能已验证(Fig 5G/H/I)
  - CD69活化↓4倍, IFN-γ↓2.6倍, 杀伤率↓3.3倍
- **临床意义**: anti-GITR + anti-TIGIT 双靶点免疫治疗

## 关键方法学贡献
1. "结构本体论" > "类型本体论" — 整合破坏的是功能社区结构，不只是消灭类型
2. 三代指标进化(CNEM→R(S)→NP) — 直接测量胜过间接代理
3. 系统性亚群破坏是普遍现象，非个例
